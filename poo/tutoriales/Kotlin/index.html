Caracteristicas

¿Que hace especial a Kotlin?

Multiplataforma

Kotlin genera archivos en bytecode por lo le permite correr en la JVM (Java Virtual Machine) y lo comvierte en un lenguaje multiplataforma.

Consiso

Kotlin esta hecho para que puedas realizar tus aplicaciones con muy poco codigo, a comparacion de Java es un 40% menos codigo para escribir.

Es NullSafety

Si!, es NullSafety, Kotlin nos permite tener seguridad al trabajar con los valores de nuestras variables sin el temor de obtener un NullPointerExceprion, no solo nos permite seguir la ejecucion aun cuando el valor es nulo, tambien nos permite asignar un valor por defecto con ayuda del operador elvis.

Kotlin es 100% interoperable con Java

Kotlin te permite no solo trabajar con la librerias de Java, tambien te permite traducir de Java a Kotlin y viseversa.
