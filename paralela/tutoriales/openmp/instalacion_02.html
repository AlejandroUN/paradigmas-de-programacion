Instalación en Linux (Ubuntu 18.04)

Read More

Paso a Paso

01. Instalación IDE CodeBlocks

Primero, es necesario ejecutar una terminal para instalar el IDE CodeBlocks, por medio de los siguientes comandos: 

                  sudo apt-get update
                  sudo apt-get install codeblocks

Después de instalar el IDE CodeBlocks, se selecciona GNU GCC como compilador por defecto. Para este sistema operativo no es necesario instalar los complementos de MingW.

02. Configuracion CodeBlocks

Es necesario habilitar la librería omp.h en el IDE CodeBlocks para poder usarla. Para ello, es necesario ejecutar CodeBlocks, Ir a Settings -> Compiler en la barra de menú, luego en la ventana que aparece, ubicarse en la pestaña Compiler settings -> Other options, para agregar la bandera -fopenmp.





Ahora es necesario ubicarse en la pestaña Compiler settings -> Linker settings para luego agregar la librería “gomp” (Escribiendo el texto gomp directamente) y después se da clic primario en el botón OK para salir de la ventana:







Después de haber realizado estos pasos, ya se tendrá OpenMP en Linux (Ubuntu 14.04) para el IDE CodeBlocks.
